Welcome to my AngularJS Application

This is the first application I have made using AngularJS and D3.js. This application will break down the time spent on this project into subject categories and display them visually with a Pie chart and a Stacked Bar chart. These elements are interactive (hover over data to get a value) and responsive to different screen sizes. It was brought up during our interview how important time-keeping is to this internship, so I decided to track my time throughout this project and use the data gathered from it to display my visualizations.

Time Spent

This bar chart gives us a time breakdown per day of what I was doing as well as how long I was working on it. But what was the total composition of the time spent on this project?

Time Proportions

Time when hovering over slices is also in minutes.

As you can see from the pie chart there is a relatively even split between the time spent on this project and the subject I was working on. This is because more time had to be dedicated towards AngularJS/D3 initially to learn the framework/library before actually getting started on the application itself.
